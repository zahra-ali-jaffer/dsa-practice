# Collections Operations Summary

## Stacks (LIFO)
**Operations:** `isEmpty()`, `size()`, `toString()`, `push(T el)`, `pop()`, `peek()`, `isFull()`, `capacity()`

## Queues (FIFO)
**Operations:** `isEmpty()`, `size()`, `toString()`, `enqueue(T el)`, `dequeue()`, `first()`, `isFull()`, `capacity()`

## Lists
**Operations:** `isEmpty()`, `size()`, `toString()`, `removeFirst()`, `removeLast()`, `remove(T el)`, `first()`, `last()`, `iterator()`

## Maps
**Operations:** `add(K key, V value)`, `reassign(K key, V value)`, `remove(K key)`, `lookup(K key)`, `keySet()`, `valueSet()`

## Sets
**Operations:** `isEmpty()`, `size()`, `toString()`, `add(T el)`, `remove(T el)`, `retainAll(Set<T> s)`, `contains(T el)`, `equals(Set<T> s)`, `isSubset(Set<T> s)`, `union(Set<T> s)`, `intersection(Set<T> s)`, `difference(Set<T> s)`, `pick()`, `removeRandom()`, `iterator()`

## MultiSets
**Operations:** `isEmpty()`, `size()`, `toString()`, `add(T el)`, `remove(T el)`, `retainAll(MultiSetADT<T> s)`, `contains(T el)`, `equals(MultiSetADT<T> s)`, `isSubset(MultiSetADT<T> s)`, `union(MultiSetADT<T> s)`, `intersection(MultiSetADT<T> s)`, `difference(MultiSetADT<T> s)`, `count(T el)`, `sum(MultiSetADT<T> other)`, `iterator()`
